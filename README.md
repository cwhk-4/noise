コントローラー、または方向キーでキャラクターを操作し、ギミックや傘を使って、ゴールを目指します。

傘は落ちてくるものからガードしたり、風を利用して飛んだりすることができます。

途中に罠(歩くスピードが減速)や攻撃もあります。

罠に当たると歩くスピードが減速します。

攻撃に当たったり、穴に落ちてしまうとチェックポイントに戻されます。

ゴールに着きましたらクリアです。
